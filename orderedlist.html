1. <html:>The root element of an HTML document. It contains all other elements and defines the document type.
  2. <head:> Contains metadata about the HTML document, such as the title, linked stylesheets, and scripts.
  3. <title>:Defines the title of the document, which appears in the browser's title bar or tab.
  4. <meta>:Provides metadata such as character set, author, and viewport settings. It doesn't have a closing tag.
  5. <link>:Used to link external resources like stylesheets or icons to the document. It also doesn't have a closing tag.
  6. <script>: Embeds or references JavaScript code in the document.
  7. <body>:Contains the content of the HTML document, such as text, images, and links.
  8. <header>Represents introductory content or a set of navigational links, usually containing headings, logos, or search forms.
  9. <nav>:Defines a section of navigation links.
  10. <h1>:Represents headings
  11. <p>: Defines a paragraph of text.
  12. <a>: Creates a hyperlink to another document or resource.
